# Ruby examples

Pure-Ruby examples for the `json` library as provided by
[go-embedded-ruby](https://github.com/go-embedded-ruby/ruby) (rbgo). Run them
with the `rbgo` interpreter:

```sh
rbgo examples/json_usage.rb
```

| File | Shows |
| --- | --- |
| [`json_usage.rb`](json_usage.rb) | `generate`, `parse` and `pretty_generate`. |

Each example is executed as-is under rbgo (`require "json"`).
